CPU;Professional;У вас установлен серверный процессор. Несмотря на высокую производительность. В некоторых проектах он может быть слабее своего десктопного аналога
CPU;No Requier;Ваш текущий процессор не соответствует минимальным требованиям большинства современных игр. Необходимо сметить процессор на более можный или поменять платформу (процессор + маринская плата)
CPU;Min Requier;Ваш текущий процессор соответствует минимальным требованиями большинства игр, однако стоит задуматься о покупке более мощного процессора
CPU;Rec Requier;Ваш текущий процессор соответствует рекомендуемым требованиями примерно половины современных игр. Смена процессора необязательна, только на ваше усмотрение
CPU;Full Requier;Ваш текущий процессор соответствует рекомендуемым требованиями большинства современных игр. Если нет рабочей (например для монтажа видео или обучения ИИ) необходимости, процессор можно не менять
GPU;Professional;Внимание! У вас установлена профессиональная видеокарта, которая не предназначена для игр. Скорее всего, вы сможете запустить игры, но не сможете с комфортом играть в них
GPU;No Requier;Ваша текущая видеокарта не соответствует минимальным требованиям большинства современных игр. Необходимо сметить видеокарту на более мощную
GPU;Min Requier;Ваша текущая видеокарта соответствует минимальным требованиями большинства игр, однако стоит задуматься о покупке более мощной видеокарты
GPU;Rec Requier;Ваша текущая видеокарта соответствует рекомендуемым требованиями примерно половины современных игр. Смена видеокарты необязательна, только на ваше усмотрение
GPU;Full Requier;Ваша видеокарта соответствует рекомендуемым требованиями большинства современных игр. Если нет рабочей (например для рендера видео или глубокого изучения ИИ) необходимости, видеокарту можно не менять
RAM;Type Memory;Ваш компьютер успользует устаревший тип памяти. Рекомендуется в скором времени сменить платформу (процессор + маринская плата) для комфортой работы и игры
RAM;No Requier;У вас установлен объём оперативной памяти, не достаточный для большинства современных игр, рекомендуется увеличить объём
RAM;Min Requier;У вас установлен объём оперативной памяти,достаточный для половины современных игр, однако его категорически не хватает для комфортной игры на высоких настройках. Желательно увеличение объёма оперативной памяти
RAM;Rec Requier;У вас установлен объём оперативной памяти,достаточный для большинства современных игр, однако недостаточно для игры на максимальных настройках. Желательно увеличение объёма оперативной памяти
RAM;Full Requier;У вас установлен объём оперативной памяти,достаточный для большинства современных игр, увеличение объёма по желанию
HARD;Type Connection;Ваш основной/самый производительный диск использует технологию HDD (запись на магнитные головки). Для более быстрой работы рекомендуется использовать SSD
HARD;Min Value;Объёма вашего диска хватает для установки почти любой игры, однако критично не будет хватать вместе с установленной на ней системой, обязательна покупка другого диска большего объёма или покупка дополнительного диска для хранения файлов и игр
HARD;Norm Value;Объёма вашего диска хватает для установки любой игры и установки системы. Для установки более 2-х игр рекомендуется покупка другого диска большего объёма или покупка дополнительного диска для хранения файлов и игр
HARD;Max Value;Объёма вашего диска хватает для установки нескольких игр и установки операционной системы. Покупка дополнительного диска по желанию
OS;Name OS;Вы используете устаревшую операционную систему Windows 7. Данная система перестала поддерживаться, поэтому рекомендуется обновиться до более поздних версий
OS;32 Bit;Внимание! Вы используете 32 битную версию системы. Данная версия не подходит для игр и некоторых профессиональных задач, поэтому рекомендуется обновиться до 64 битной версии
OS;64 Bit;Вы используете 64 битную версию системы, подходящую для игр. Однако, если производительность вашей системы на низком уровне, рекомендуется либо улучшить конфигурацию, либо сменить версию операционной системы на более лёгкую
